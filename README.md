# AWS 3-Tier Architecture

## Project Overview

This project demonstrates the deployment of a secure, scalable, and highly available 3-tier web application on Amazon Web Services (AWS). The architecture includes separate Web, Application, and Database tiers with Load Balancers, Amazon RDS, and 
Auto Scaling to ensure reliability and performance.

## AWS Services Used

- Amazon VPC
- Amazon EC2
- Amazon RDS (MySQL)
- Application Load Balancer
- Auto Scaling
- NAT Gateway
- Security Groups
- AWS Systems Manager
